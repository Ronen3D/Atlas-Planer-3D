
# Atlas Planer 3D Module

A 3D planning and visualization tool for the Aviad Atlas project.

## Project Structure

```
src/
├── index.html          # Entry point
├── sample.js           # Sample implementation
├── ap3d/               # Core library
│   ├── Hooks.ts        # React hooks
│   ├── index.ts        # Main exports
│   ├── Item.ts         # Item class
│   ├── Items.ts        # Items collection
│   ├── Manager.ts      # Manager class
│   ├── Scene.ts        # Scene management
│   └── Terrain.ts      # Terrain handling
└── assets/
    ├── 3DModels/       # 3D model files
    ├── images/         # Image assets
    └── Data/
        └── library.json # Data library
```

## Getting Started

1. Open `index.html` in a web browser
2. Refer to `sample.js` for implementation examples

## Features

- 3D scene management
- Terrain visualization
- Item placement and management
- React hooks integration

## Development

Built with TypeScript for type safety and maintainability.
